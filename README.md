## Task 1: Create a Dictionary of Student Marks
This program manages student marks using a dictionary lookup system.
It stores predefined student names and marks, accepts user input for a name, and displays the marks in the format "Aman's marks: 85" if found.
If the student does not exist in the dictionary then it displays "Student not found.".

## Task 2: Demonstrate List Slicing 
This program demonstrates list manipulation with range, slicing, and reversal.
It create a list of numbers from 1 to 10 then extracts the first five numbers and prints both.
Also reverses first five numbers and print it.
